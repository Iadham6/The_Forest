# The_Forest
A habit tracker website that track the progress of yourself
The Forest is a full-stack habit tracker and focus web application designed to help users build positive habits and stay consistent over time.
The application allows users to create daily habits, track their progress, and visualize their achievements in a simple and motivating way.

Project Goal

The main goal of this project is to provide users with a structured and user-friendly platform to:

Stay consistent with daily habits

Improve focus and productivity

Monitor progress and personal growth over time

From an academic perspective, the project aims to demonstrate the full software development life cycle (SDLC), including requirements analysis, design, implementation, and version control using GitHub.

Core Features

- Authentication module (in progress)
Register, login, and logout functionality with secure access.

Habit Management
Create, edit, and delete habits with custom names and frequencies.

Daily Check-In
Mark habits as completed each day.

Streak Tracking
Automatically calculate and display habit streaks.

Dashboard
View a summary of habits, daily progress, and streak statistics.

Pomodoro Focus Timer
Start, pause, reset, and complete Pomodoro sessions (e.g., 25/5).
